Send emails to your website administrators based on pre-defined conditions.

== Description ==

Admin Emails let's you setup automatic emails when a user earns a certain element to inform to your website administrators on exact moment you want.

In just a few minutes, you will be able to build a personalized email to being sent on relevant events like when a user reaches a points balance, unlocks an achievement or reaches a rank among others.

In addition, Admin Emails includes a huge list of tags to let you display relevant information about the user who earned something to your administrator like the user current points balance, last achievements earned and current rank of any type.

= Features =

* Send customizable emails to your website administrator when a users earns a certain element.
* Set the condition you want: reach a points balance, unlock an achievement, unlock any achievement of type, unlock all achievements of type or reach a rank.
* Tags per type to inform administrator about the user current points balance, last achievements earned and current rank of any type.
* Ability to set the roles you want to send emails.
* Ability to send a test email to yourself.

== Installation ==

= From WordPress backend =

1. Navigate to Plugins -> Add new.
2. Click the button "Upload Plugin" next to "Add plugins" title.
3. Upload the downloaded zip file and activate it.

= Direct upload =

1. Upload the downloaded zip file into your `wp-content/plugins/` folder.
2. Unzip the uploaded zip file.
3. Navigate to Plugins menu on your WordPress admin area.
4. Activate this plugin.
